# Sowing the Seeds for Practical Multimodal Crop Yield Prediction: A Systematic Review with Recommendations for Future Work
## [Natalie Hollain](https://www.au.dk/en/natalie.hollain@agro.au.dk/), [Sheng Wang](https://scholar.google.com/citations?user=BxTGuO8AAAAJ&hl=en), [René Gislum](https://scholar.google.com/citations?user=n6jQps0AAAAJ&hl=en)
### Institute for Agroecology, Aarhus University, Denmark

This repository contains the data and code for the review paper **"Sowing the Seeds for Practical Multimodal Crop Yield Prediction: A Systematic Review with Recommendations for Future Work"**. The contents are as follows:

- `systematic_literature_review.csv` contains an analysis of 85 papers. This includes information about the publication year, studied crops, data modalities, et cetera. In total, there are 22 columns.
- `systematic_literature_review.ipynb` contains the code to analyze the 85 papers. This notebook creates a folder, `images/`, in which all the figures are stored as both a PDF and PNG.
